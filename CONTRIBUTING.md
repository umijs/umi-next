# Umi Contributing Guide

Loving Umi and want to get involved? Thanks!

## Set up

```bash
# Clone this repo
$ git clone git@github.com:umijs/umi-next.git
$ cd umi-next
# Install Dependencies
$ pnpm i
# Compile Src
$ pnpm build
# Start dev server for boilerplate example
$ cd examples/boilerplate
$ pnpm dev
```

## Test

```bash
$ pnpm test
```
